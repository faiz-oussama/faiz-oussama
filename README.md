## 🔥 Current Project

I'm currently building a **Full-stack ticketing platform** that bridges the gap between event organizers and clients:

- 🌐 **Frontend**: React.js, Redux, Tailwind CSS
- ⚙️ **Backend**: Spring Boot, Spring Security, Spring Data JPA
- 🛠️ **DevOps**: CI/CD pipeline, Docker, Kubernetes
- 📊 **Database**: MySQL, Redis for caching
- 🔐 **Authentication**: JWT, OAuth2
